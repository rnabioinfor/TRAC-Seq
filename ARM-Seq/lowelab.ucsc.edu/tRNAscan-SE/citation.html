Credits and Citations

The tRNAscan-SE Search Server was created and is maintained by Todd Lowe and Patricia Chan.

Citations

If you use this web server to predict tRNAs, please cite:

    Lowe, T.M. & Chan, P.P. tRNAscan-SE On-line: Search and Contextual Analysis of Transfer RNA Genes. (Submitted)
    Lowe, T.M. & Eddy, S.R. (1997) tRNAscan-SE: A program for improved detection of transfer RNA genes in genomic sequence. Nucl. Acids Res. 25: 955-964.

If you find this resource helpful and use it in your research, please help support its funding by writing a quick email to Todd (lowe@soe.ucsc.edu) describing how it has been useful to you. Letters like these help bolster the case for future funding. (many thanks! -Todd)

tRNAscan-SE employs
  1. Infernal: the RNA covariance similarity search package (Nawrocki & Eddy, Bioinformatics 29: 2933-2935, 2013),
  2. a modified, optimized version of tRNAscan v1.3 (Fichant & Burks, J. Mol. Biol. 220: 659-671, 1991),
  3. a new implementation of a multistep weight matrix algorithm for identification of eukaryotic tRNA promoter regions (Pavesi et al., Nucl. Acids Res. 22: 1247-1256, 1994),
  4. the RNA covariance analysis package Cove v.2.4.2 (Eddy & Durbin, Nucl. Acids Res. 22: 2079-2088, 1994)
